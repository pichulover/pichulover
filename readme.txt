Program codes and submissions for Pikalover

submission_trial_1.csv is generated by MSBD5001_trial_1.ipynb.
submission_trial_2.csv is generated by MSBD5001_trial_2.ipynb.


The programs are written by Python 3 with the following packages:
numpy, pandas, sklearn.ensemble, sklearn.model_selection

mean_squared_error from sklearn.metrics is not used in the program.


Required input files for executing the program:
samplesubmission.csv
test.csv
train.csv

After the program is exeecuted, the file submission.csv will be the output file.
